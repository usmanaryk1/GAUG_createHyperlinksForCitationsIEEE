How to setup this:
1. put the whole folder ontime-web_frontend to tomcat/webapps

2. put project ontime-web_backend to tomcat/webapps

3. start tomcat to run frontend and backend together

4. visit url http://localhost:8080/ontime-web_frontend/
5. example urls: 
   1) http://localhost:8080/ontime-web_frontend/#/app/add_patient_tab_1
   2) http://localhost:8080/ontime-web_frontend/#/app/add_patient_tab_2
   3) http://localhost:8080/ontime-web_frontend/#/app/add_patient_tab_3
   4) http://localhost:8080/ontime-web_frontend/#/app/add_patient_tab_4
   5) http://localhost:8080/ontime-web_frontend/#/app/add_patient_tab_5

   6) http://localhost:8080/ontime-web_frontend/#/app/view_patient_tab_1
   7) http://localhost:8080/ontime-web_frontend/#/app/view_patient_tab_2
   8) http://localhost:8080/ontime-web_frontend/#/app/view_patient_tab_3
